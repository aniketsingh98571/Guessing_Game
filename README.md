# Guessing_Game
A Simple Guessing game built using HTML, CSS and Javascript. In this game computer will generate a random number between 1 to 100 and on the otherside
user will need to guess the number, 
```
if(number matches){
  "You won"
}
else{
  "Try again"
}
```
